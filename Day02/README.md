## ✨ 배운 점
- `for` 반복문은 횟수가 명확할 때 사용, `while`은 조건 기반일 때 더 자연스러움
- 중첩 반복문의 구조가 처음엔 헷갈렸지만 구구단 출력 문제로 익숙해짐
- `%` 연산자를 통해 짝수/홀수 판별 가능
- `Scanner` 사용 시 여전히 `.close()`를 잊지 말아야 경고 없이 마무리 가능

## ❗️주의하거나 반복 학습할 점
- 중첩 반복문에서 인덱스 변수 헷갈리지 않게 주의 (`i`, `j` 등)
- `sum += i`처럼 누적 패턴은 다른 문제에도 자주 쓰이므로 확실히 익혀두기
